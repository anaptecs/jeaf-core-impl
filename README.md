# JEAF Core Impl README #

This repository contains the source code of JEAF Core Implementation. JEAF Core Impl provides implementation of JEAF Core API such as service channel implementation for
different runtime environments as JSE, Servlet Containers or JEE Application Servers.. 

## Links ##
For further information please refer to:

* [JEAF Core](https://anaptecs.atlassian.net/l/c/2V8CjTAe)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.core/jeaf-core-impl-project)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
